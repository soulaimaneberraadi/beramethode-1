Synchroniser ce PC vers le cloud

Cette page lit les données du serveur Express local (database.sqlite) et les pousse vers Supabase sous votre compte. Ouvrez l'application sur mobile avec le même e-mail — les données apparaîtront.
À exécuter uniquement depuis le PC où tourne npm run dev:app.

Le mot de passe n'est utilisé que dans ce navigateur pour s'authentifier auprès de Supabase. Il n'est ni stocké, ni envoyé au serveur Express.
